# **App Name**: Planify

## Core Features:

- Category Selection: Allow users to select between 'Academics' and 'Personal' categories to tailor the app's functionality.
- Sub-Category Selection: Enable users to further specify their role (e.g., 'Student', 'Professor', 'Management') within their selected category.
- Questionnaire: Present users with a series of questions specific to their selected category and sub-category.
- Data Analysis and Scheduling: Analyze user responses from the questionnaire using an AI tool to create a personalized schedule.

## Style Guidelines:

- Primary color: Teal (#008080) - reflects the teal color of the PLANIFY title logo, evoking a sense of calm and organization.
- Background color: Light beige (#F5F5DC) - a desaturated hue similar to the teal, providing a soft and unobtrusive background.
- Accent color: Light cyan (#E0FFFF) - An analogous hue to teal for interactive elements such as buttons and links.
- Body and headline font: 'PT Sans' for clear, readable text throughout the application.
- Simple layout with clear call-to-action buttons to drive category selections.
- Use icons on the main category screen buttons. Icons related to academics (graduation cap, books) and personal (calendar, clock).
- Subtle animations when transitioning between pages of the questionnaire.
